## documentation.json Configuration

The main configuration file controls your entire documentation structure.

<CodeGroup tabs="json,yaml">
  ```json
  {
    "name": "My Documentation",
    "initialRoute": "introduction",
    "colors": {
      "light": { "brand": "#3B82F6" },
      "dark": { "brand": "#60A5FA" }
    }
  }
  ```
  ```yaml
  name: My Documentation
  initialRoute: introduction
  colors:
    light:
      brand: "#3B82F6"
    dark:
      brand: "#60A5FA"
  ```
</CodeGroup>

## Advanced Settings

Fine-tune performance and functionality.

<Callout kind="alert">Advanced settings can impact site performance if not configured properly.</Callout>

| Setting | Description | Default |
|---------|-------------|---------|
| Caching | Enable caching for faster page loads | Enabled |
| Compression | Compress assets for better performance | Enabled |
| Analytics | Integrate with analytics platforms | Disabled |
| Search | Enable full-text search functionality | Enabled |

## Environment Variables

Use environment variables for sensitive configuration.

<CodeGroup tabs="bash,powershell">
  ```bash
export KOTTLIN_TRI_API_KEY="your-api-key"
export KOTTLIN_TRI_ENVIRONMENT="production"
  ```
  ```powershell
$env:KOTTLIN_TRI_API_KEY="your-api-key"
$env:KOTTLIN_TRI_ENVIRONMENT="production"
  ```
</CodeGroup>
